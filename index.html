처음 한 번만: Firebase 연결 설정

이 파일의 코드 안에 Firebase 설정이 아직 입력되지 않았습니다.

1. console.firebase.google.com → 프로젝트 추가 (애널리틱스는 꺼도 됩니다)
2. 빌드 → Realtime Database → 데이터베이스 만들기 → 잠금 모드로 시작
3. 규칙 탭에 전달받은 보안 규칙 붙여넣고 게시
4. 프로젝트 설정(톱니) → 내 앱 → 웹 앱(</>) 추가 → firebaseConfig 복사
5. 이 HTML 파일을 열어 ★ firebaseConfig 붙여넣는 곳 ★ 자리에 붙여넣기
6. GitHub Pages에 업로드 후 두 분 폰에서 접속
